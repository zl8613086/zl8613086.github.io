约瑟夫算法

发表于 2016-05-12   |   分类于 算法   |  

问题来源

    据说著名犹太历史学家 Josephus有过以下的故事：在罗马人占领乔塔帕特后，39 个犹太人与Josephus及他的朋友躲到一个洞中，39个犹太人决定宁愿死也不要被敌人抓到，于是决定了一个自杀方式，41个人排成一个圆圈，由第1个人开始报数，每报数到第3人该人就必须自杀，然后再由下一个重新报数，直到所有人都自杀身亡为止。然而Josephus 和他的朋友并不想遵从。首先从一个人开始，越过k-2个人（因为第一个人已经被越过），并杀掉第k个人。接着，再越过k-1个人，并杀掉第k个人。这个过程沿着圆圈一直进行，直到最终只剩下一个人留下，这个人就可以继续活着。问题是，给定了和，一开始要站在什么地方才能避免被处决？Josephus要他的朋友先假装遵从，他将朋友与自己安排在第16个与第31个位置，于是逃过了这场死亡游戏。
    约瑟夫问题简单来说就是一个循环队列，从第一个人循环报数，报第M的出列，出列的下一个人重新报1开始，一直到最后出列只剩一个。

问题解法

1.

    我们可以用程序模拟这个过程，这里我们用循环链表来模拟这个过程，假设有30个人。

public class Josephus {
class node{
    int val;
    node pre=null;
    node next=null;
    public node(int val){
        this.val=val;
    }
}
public node init(int n){
    if(n<=0)
        return null;
    node head=new node(1);        
    node pre=head;
    for(int i=1;i<n;i++){
        node cur=new node(i+1);
        pre.next=cur;
        cur.pre=pre;
        pre=cur;
    }
    pre.next=head;
    head.pre=pre;
    return head;

}

public int select(node head,int m){
    if(head==null){
        return 0;
    }
    int n=1;
    node cur=head.next;
    while(cur!=head&&cur!=null){
        n++;
        cur=cur.next;
    }
    if(cur==null)
        return 0;
    int s=0;
    cur=head.pre;
    while(n>1){
        cur=cur.next;
        s++;
        if(s==m){
            s=0;
            cur.pre.next=cur.next;
            cur.next.pre=cur.pre;
            n--;
        }
    }

    return cur.next.val;
}
public static void main(String[] args) {
    Josephus js=new Josephus();
    node head=js.init(41);
    int i=js.select(head, 3);
    System.out.println(i);
}

}

    当然也可以用数组去模拟，出列的位置置0。

2.

    我们假设标号从0开始，第一个人（编号一定是m) 出列之后，剩下的n-1个人组成了一个新的约瑟夫环（以编号为k=(m) mod n的人开始）：

k k+1 k+2 … n-2,n-1,0,1,2,… k-2

并且从k开始报1。
我们把他们的编号做一下转换：

k –> 0

k+1 –> 1

k+2 –> 2

…

…

k-2 –> n-2

    变换后就完完全全成为了（n-1）个人报数的子问题，假如我们知道这个子问题的解：例如x是最终的胜利者，那么根据上面这个表把这个x变回去不刚好就是n个人情况的解吗？！！变回去的公式很简单，相信大家都可以推出来：x’=(x+k) mod n。因为k=(m)%n;x’=(x(m)%n)%n=(x+m)%n。
如何知道（n-1）个人报数的问题的解？对，只要知道（n-2）个人的解就行了。（n-2）个人的解呢？当然是先求（n-3）的情况 —- 这显然就是一个倒推问题！好了，思路出来了，下面写递推公式：
令f表示i个人玩游戏报m-1也就是第m个人退出最后胜利者的编号，最后的结果自然是f[n]
递推公式
f[1]=0;
f[i]=(f[i-1]+m) mod i; (i>1）
有了这个公式，我们要做的就是从1-n顺序算出f的数值，最后结果是f[n]。由于标号一般从1开始 结果最后加1：f+=1；

int JosephusProblem_Solution(int n, int m)  
{  
if(n < 1 || m < 1)  
return -1;  

int f=0;

for(int i = 2; i <= n; i++)  
f = (f + m) % i; //按递推公式进行计算  
int result = f;         
return result;  
}
